# Kiosk 주문 관리 시스템 솔루션

무인 주문/주방관리/대기관리 시스템과 결제/프린터/카메라 등 주변기기 연동을 포함한 키오스크 솔루션 프로젝트입니다.

## 프로젝트 개요
- 회사 : (주)로보러스
- 목적: 기존의 무인 결제 시스템에 Deep-Learning을 통한 얼굴 인식 프로그램 구현 하여 매장 무인 주문 프로세스 자동화 및 운영 효율화
- 기간: 2018.03 - 2019.07
- 역할: 서버/DB 연동 및 시스템 통합 개발

## 주요 역활
- Node.js 웹 서버와 MongoDB 연동 기반 주문 관리 기능 구현 담당
- MySQL 기반 로컬 운영 데이터 처리 구성
- Python Django API를 활용한 기능 연동(프로젝트별)
- C# WPF을 이용한 FrontEnd UI/UX 구현 보조
- 결제기(PostLink), 프린터, 카메라 등 주변기기 통신 구현
- 미국 매장 운영 환경(AmericanDeli) 적용 경험
- 무인 결제 시스템의 새로운 로봇 버전의 상황 별 모터 및 LED 구동 컨트롤 구현

## 성과
### 1. [시스템 통합 및 상용화]  하드웨어 제어와 미국 현지 매장 도입

- 개선 성과: 결제 단말기(PostLink), 영수증 프린터, 카메라 등 다양한 주변기기와 Node.js 백엔드 간의 통신 로직을 완벽하게 통합했습니다. 딥러닝 얼굴 인식 모델을 결제 프로세스에 매끄럽게 연결하여, 실제 미국 현지 매장(AmericanDeli) 및 전시 환경에서 무인 주문 시스템을 성공적으로 상용화했습니다.

### 2. [안정성 강화] 모듈화를 통한 하드웨어 장애 격리

- 개선 성과: 현장에서 흔히 발생하는 장비 에러가 전체 시스템 다운으로 이어지지 않도록, 주변기기(결제기, 프린터, 카메라)와의 통신 로직을 독립적인 모듈로 철저히 분리했습니다. 특정 장비의 통신 지연이나 장애 발생 시 즉각적으로 대응 및 복구가 가능하도록 결합도를 낮추어, 오프라인 매장 운영 이슈를 대폭 감소시켰습니다.

### 3. [아키텍처 설계] 목적에 최적화된 시스템 구축

- 개선 성과: 매장 내 빠른 처리가 필요한 로컬 데이터는 MySQL로, 유연한 구조의 주문 데이터는 MongoDB로 분리 구성하여 데이터 계층을 최적화했습니다. 또한 딥러닝 연동을 위한 Python(Django) API와 메인 로직인 Node.js를 결합하는 등, 프로젝트의 요구사항에 맞춰 최적의 언어와 데이터베이스를 조합하는 유연한 아키텍처 설계 역량을 확보했습니다.

### 4. [제어 확장] 무인 결제 로봇 인터페이스 동기화 구현

- 개선 성과: 정적인 키오스크를 넘어선 새로운 무인 로봇 버전 시스템에 참여하여, 사용자의 주문 상황(결제 대기, 완료 등)과 동기화되어 작동하는 물리적 모터 및 LED 구동 컨트롤 로직을 추가로 구현했습니다.

## 기술 스택
- Node.js, Python, Django
- MongoDB, MySQL
- C#, WPF
- Device Integration (결제기, 프린터, 카메라)
