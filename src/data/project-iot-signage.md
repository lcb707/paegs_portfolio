# IoT 전광판 관리 플랫폼

사용자가 원하는 영상을 직접 제작하고, 스케줄을 설정하여 전광판에 송출할 수 있는 IoT 기반 플랫폼.
일본 세븐일레븐, 삼성전자 캠퍼스, 공공기관 등에 적용된 IoT 전광판 관리 서버 프로젝트로 실시간 데이터를 연동 하여 이미지 및 영상 송출(날씨, 교통, 뉴스, 주차 상태, 버스 정류장 정보)

## 프로젝트 개요
- 회사 : (주)이성
- 목적: 영상 제작/스케줄링/송출을 통합 관리하는 IoT 기반 전광판 플랫폼 구축
- 기간: 2019.12 - 2023.03
- 역할: 서버 관리자(백엔드 전반 개발 및 운영)

## 주요 역활
- 백엔드 시스템 개발
  - Node.js(TypeScript) 기반 REST API 서버 개발 및 유지 보수 및 신규 기능 개발
  - Python Flask 기반 외부 데이터 크롤링 및 연동 (날씨/교통/뉴스 등)
  - Golang 기반 영상 인코딩 서버 유지 보수 및 인코딩 옵션 관리
  - Node.js 기반 MQTT 통신 서버(Broker) 개발 및 장비 간 송신 구조 개선
  
- 데이터 인프라 구축 및 운영
  - MySQL 데이터베이스 설계, 쿼리 최적화, 백업 관리
  - Redis 캐시 시스템 개선 및 구현
  - AWS(EC2/S3/RDS), Docker, Nginx 기반 운영 환경 관리
  - NAS 서버를 활용한 테스트 및 백업 환경 관리

- 시스템 모니터링 및 운영
  - 서비스 장애 대응 및 로그 분석
  - 성능 개선을 위한 모니터링 시스템 구현
  - GitLab 기반 코드 관리 및 ApiDoc을 통한 RESTful API 문서화

## 성과
### 1. [성능 최적화] 제한된 단일 서버 환경에서의 트래픽 병목 해소 및 최적화

- 문제: 단일 인스턴스로 운영되는 Python Flask 기반 외부 데이터 서버 환경에서, 대량의 크론잡(배치 작업)과 복잡한 DB 쿼리를 동반하는 사용자 API 요청이 동시간대에 맞물리며 uWSGI 프로세스 고갈로 인한 전체 API 응답 지연 및 타임아웃 오류 발생.

- 해결 및 성과:
  - 비동기 워커 분리: Celery와 Redis 기반의 메시지 큐를 도입하여, 이미지 생성 등 무거운 작업들을 백그라운드 Worker 프로세스로 비동기 처리하도록 구조를 개편해 메인 API 응답 경로를 확보함.
  - 쿼리 최적화 및 캐싱: 한 번의 API 요청에서 발생하는 다량의 중복 쿼리 호출을 최적화하고, 적극적인 Redis 캐시 전략을 적용하여 DB 부하를 획기적으로 경감시킴.
  - 서버 튜닝: uWSGI 프로세스/스레드 설정을 현재 서버 자원(CPU/RAM)에 맞춰 세밀하게 튜닝하여, 추가적인 서버 증설(비용 지출) 없이 타임아웃 에러를 완전히 해소하고 안정적인 응답 속도를 확보함.

### 2. [아키텍처 고도화] 비동기 스케줄링 및 IoT 디바이스 통신 신뢰도 향상

- 해결 및 성과:
  - 대용량 파일 인코딩 등의 리소스 집약적인 작업 역시 Job 스케줄링 패턴을 최적화하여 안정적으로 큐잉되고 처리되도록 백엔드 구조를 고도화함.
  - IoT 장비 간 MQTT 통신 및 메시지 송수신 흐름을 재설계하여, 장비 연동 시 발생하던 송출 실패 및 지연 케이스를 획기적으로 감소시키고 운영 가시성을 확보함.

### 3. [DevOps & 인프라] 운영 자동화 및 실시간 장애 대응 체계 구축

- 해결 및 성과:
  - 자체 모니터링 알림 시스템 구축: 서버 에러나 이상 징후 감지 시 텔레그램 및 이메일로 즉시 알림이 발송되는 파이프라인을 자체 개발하여, 선제적인 장애 대응 골든타임을 확보함.
  - 수동으로 진행되던 HTTPS 인증서 갱신 작업을 스크립트 기반으로 자동화하여 인증 만료로 인한 서비스 접근 장애 리스크를 원천 차단함.
  - MySQL DB 백업/복구 프로세스와 NAS 기반 테스트 백업 환경을 구축하고, 정기적인 에러 로그 분석 루틴을 정착시켜 안정적인 인프라 운영 표준을 확립함.

### 4. [개발 생산성] 테스트 환경 구축 및 문서화를 통한 협업 효율 극대화

- 해결 및 성과:
  - 신규 비즈니스 요구사항에 맞춘 RESTful API를 지속적으로 확장 개발함.
  - Postman을 활용하여 부서 간 통합 API 테스트 환경을 구축하고, ApiDoc을 통한 API 명세서 자동화 및 GitLab 코드 리뷰 문화를 정비함.
  - 프론트엔드 및 클라이언트 팀과의 소통 비용을 최소화하고, 신규 기능 반영 및 이슈 대응 속도를 크게 단축함.

## 기술 스택
- TypeScript, Node.js, Python, Flask
- MySQL, Redis, MQTT
- AWS EC2/S3/RDS, Docker, Nginx
